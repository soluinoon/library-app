# library-app
도서대출 웹 어플리케이션을 구현해보는 레포지토리

## Overview
도서대출 웹 어플리케이션을 구현해보며 스프링 웹 프로젝트의 전반적인 흐름을 알아보는 레포지토리 입니다.

### 목표
1. 3 Layered Architecture 적용
2. JDBC -> JPA 전환
3. 실제 서버 DNS 구입해서 적용
